## 概述
这里的双指针不是说有两个指针的, 
而是更狭义的, 双指针维护区间问题

特点就是 : 具有单调性, l向右移动 r也只能向右移动
(对于任意li <= lj , ri <= rj)
![[双指针特性.excalidraw]]

## 子问题
乘积(和)小于k 的区间计数
最长连续不重复子序列




## 代码
```cpp
for(int i = 1, j = i - 1; i < n; i++) { // 注意, j = i - 1
	while(j + 1 < n && check(a[j + 1])) { 
		j++;
		upd(a[j], 1); // 信息统计上去
	}
	// do i , j 
	upd(a[i], -1); // 信息清除
}
```
